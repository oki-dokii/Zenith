# Netflix AI Assistant - Demo Script

## Overview
This is a 30-60 second demo script for screen recording. The demo shows the AI assistant helping users find movies within the Netflix desktop app.

---

## Setup Checklist

Before recording:

- [ ] Netflix desktop app installed and logged in
- [ ] Netflix AI Assistant running (`python -m netflix_assistant.main`)
- [ ] Screen recording software ready
- [ ] Netflix window visible on screen
- [ ] Resolution set for clear visibility (1920x1080 recommended)

---

## Demo Script

### Scene 1: Introduction (5 seconds)

**Narration**: "This is the Netflix AI Assistant - an intelligent search helper for the Netflix desktop app."

**Action**: Show the Netflix app on screen, cursor hovering near the search icon.

---

### Scene 2: Opening Search (5 seconds)

**Narration**: "Let me show you how it works."

**Action**: 
1. Click on the Netflix search icon
2. Search bar becomes active

---

### Scene 3: Typing AI Query (10 seconds)

**Narration**: "When I type 'AI:' followed by what I'm looking for..."

**Action**:
1. Start typing in the search bar
2. Type: `AI: horror movies`
3. As you type "AI:", pause briefly
4. Continue typing "horror movies"

---

### Scene 4: Suggestions Appear (10 seconds)

**Narration**: "The AI assistant instantly shows me curated suggestions."

**Action**:
1. The overlay appears below the search box
2. Show the list of horror movies with ratings
3. Hover mouse over the suggestions briefly

---

### Scene 5: Navigation (10 seconds)

**Narration**: "I can navigate using arrow keys or mouse..."

**Action**:
1. Press Down arrow a few times (selection moves)
2. Press Up arrow once
3. Highlight moves visually

---

### Scene 6: Selection (10 seconds)

**Narration**: "And when I press Enter, my selection is automatically searched in Netflix."

**Action**:
1. Press Enter to select a movie (e.g., "The Conjuring")
2. Overlay disappears
3. Netflix search results appear with the movie

---

### Scene 7: Conclusion (5 seconds)

**Narration**: "Smart, simple, and seamless."

**Action**:
1. Show the search results
2. Maybe click on the movie to show it works

## Recording Tips

1. **Smooth typing**: Type at a steady, readable pace
2. **Pause on overlay**: Give viewers 2-3 seconds to read suggestions
3. **Show keyboard shortcuts**: Make arrow key navigation visible
4. **Clean desktop**: Hide other apps/notifications
5. **Netflix account**: Use a real account so results load

---

## Troubleshooting During Demo

**Overlay doesn't appear?**
- Ensure you typed `AI:` with the colon
- Check that Netflix search is focused
- Verify the assistant is running

**Selection doesn't trigger search?**
- Make sure Netflix window is still active
- Try clicking into Netflix search box first

---

## Post-Recording

After recording:

1. Trim any dead time
2. Add captions if needed
3. Export at 1080p for clarity
4. Keep total length under 60 seconds
